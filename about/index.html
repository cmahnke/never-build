Kolophon

Diese Seite wurde mit Hugo generiert.

Als Schriftarten kommen Tektur und zum Einsatz.

Die folgenden JavaScript und CSS Frameworks werden benutzt:

  • jQuery
  • normalize.css
  • Stickybits
  • JS-Cookie
  • three.js

Die Nachbearbeitung der Digitalisate wurde mit ScanTailor Advanced durchgeführt.

Für die Darstellung der Digitalisate von Einzelbildern wird OpenLayers verwendet. Die jeweils angezeigten Ausschnitte wurden mit libvips erzeugt.

Die Seite wird von GitHub gehosted.
